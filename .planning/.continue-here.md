## What's next

**Phase 45: Metadata Compatibility** -- needs /gsd-discuss-phase 45

### Problem
urknall reports that on his WiiM Ultra, OGG passthrough audio plays but track metadata and cover art do NOT display. The root cause (per his analysis in GH #96 comment 4857569938): when canDirectStream() returns the HTTP daemon URL (http://host:port/track/ID), LMS may lose the logical spoton:// URL as the canonical track identity. Metadata lookups, artwork, Track Info menus look for spoton://track:ID but find http://host:port/track/ID.

### urknall's proposed solution (summarized)
Keep spoton:// as the canonical LMS track URL. Treat daemon HTTP URLs as internal transport only. Add fallback canonicalization in getMetadataFor(), getIcon(), trackInfoMenu(), parseDirectHeaders(). Follow Spotty's "stable logical URL" model.
