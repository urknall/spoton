    Erstelle Helper.pm mit package Plugins::SpotOn::Helper.

    Imports: strict, File::Spec::Functions (qw(catdir)), JSON::XS::VersionOneAndTwo, Slim::Utils::Log, Slim::Utils::Prefs.

    Konstanten:
    - HELPER => 'spoton'
    - MIN_BINARY_VERSION => '1.0.0'

    Package-Variablen (my):
    - $prefs = preferences('plugin.spoton')
    - $log = logger('plugin.spoton')
    - ($helper, $helperVersion, $helperCapabilities) — alle initial undef

    Sub init():
    - aarch64-Fallback: Wenn nicht ISWINDOWS und nicht ISMAC und osArch =~ /^aarch64/i, dann addFindBinPaths(catdir(Plugins::SpotOn::Plugin->_pluginDataFor('basedir'), 'Bin', 'armhf-linux'))
    - Prefs-Change-Listener auf 'binary' Key (wenn nicht main::SCANNER): setzt $helper, $helperVersion, $helperCapabilities auf undef

    Sub get():
    - Zuerst custom binary aus prefs('binary') pruefen: wenn gesetzt und $helper noch undef, helperCheck aufrufen
    - Dann _findBin mit helperCheck-Callback aufrufen
    - Warn-Log wenn kein Binary gefunden
    - Rueckgabe: wantarray ? ($helper, $helperVersion) : $helper

    Sub helperCheck($candidate, $check_ref, $dontSet):
    - Shell-Aufruf: sprintf('%s -n "SpotOn" --check', $candidate), Backticks mit 2>&1
    - Regex: /^ok spoton v([\d\.]+)/i (KRITISCH: 'spoton' nicht 'spotty')
    - Mindestversion: _versionCompare($version, MIN_BINARY_VERSION) < 0 -> warn + return 0
    - Wenn nicht $dontSet: $helper, $helperVersion, $helperCapabilities setzen
    - JSON aus zweiter Zeile: $check =~ /\n(.*)/s, dann eval { from_json($1) } || {}

    Sub _findBin($checkerCb, $customFirst):
    - Kandidaten-Liste: (HELPER) — also ('spoton')
    - Auf Unix + x86_64: push HELPER . '-x86_64' als Kandidat
    - Custom-Override: unshift HELPER . '-custom'
    - Loop: Slim::Utils::Misc::findbin($name), decodeExternalHelperPath, -f && -x check, dann $checkerCb

    Sub getCapability($class, $key):
    - Return $helperCapabilities->{$key} wenn definiert, sonst undef

    Sub getVersion($class):
    - get() aufrufen wenn $helperVersion undef, dann $helperVersion returnen

    Sub version($class):
    - Alias fuer getVersion

    Sub _versionCompare($v1, $v2):
    - Split auf '.', numerischer Vergleich jeder Komponente
  </action>
