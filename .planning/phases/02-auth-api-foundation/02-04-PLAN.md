       - Check if $paramRef->{removeAccount} is set (value = accountId to remove).
       - Call TokenManager->removeAccount($removeId).

       Account switch:
       - Check if $paramRef->{switchAccount} is set (value = accountId to activate).
       - Set $prefs->set('activeAccount', $accountId).

       - $paramRef->{accounts} = $prefs->get('accounts') || {}
       - $paramRef->{activeAccount} = $prefs->get('activeAccount') || ''
