## Fixed Issues

### CR-01 + WR-04: _killOrphanedProcesses kann aktive Transkodierungsprozesse toeten / pkill Race-Condition

**Files modified:** `Plugins/SpotOn/Plugin.pm`
**Commit:** 866e50f
**Applied fix:** Restricted `isPlaying()` check to only Spotify-playing clients (checking `$song->currentTrack()->url` for `spotify://` prefix). Non-Spotify playback no longer blocks orphan cleanup. Replaced blanket `pkill -f` with targeted `pgrep -f` + per-PID `kill 'TERM'` that skips PIDs belonging to active transcoding processes (tracked via `$song->transcodeProcess()`). This addresses both the false-positive (non-Spotify blocking cleanup) and false-negative (killing active transcoding) failure modes, and eliminates the WR-04 race condition by never killing actively-used PIDs.

### WR-01: Album-Name fehlt im Metadata-Cache fuer Album-Track-Kontext

**Files modified:** `Plugins/SpotOn/Plugin.pm`
**Commit:** d3af4ce
**Applied fix:** Four changes: (1) `_albumItem` now passes `albumName => $album->{name}` in passthrough for pagination pages. (2) `_albumFeed` extracts `$albumName` from passthrough. (3) Both call sites of `_albumTrackItem` (initial and paginated) pass `$album->{name}` / `$albumName` as 5th parameter. (4) `_albumTrackItem` signature updated to accept `$albumName`; the hardcoded `my $albumName = ''` replaced with `$albumName //= ''` fallback. NowPlaying metadata cache now contains the correct album name for album-context tracks.

### WR-02: Bitrate-Regex zu restriktiv

**Files modified:** `Plugins/SpotOn/Plugin.pm`
**Commit:** 3417015
**Applied fix:** Changed `--bitrate \d{2,3}` to `--bitrate \d+` so the regex matches any numeric bitrate value, preventing silent substitution failures on unexpected values.

### WR-03: Unreachable Code nach process::exit() in Mode::Check

**Files modified:** `librespot-spoton/src/main.rs`
**Commit:** 5188f48
**Applied fix:** Added `process::exit(0)` at the end of the `Mode::Check` match arm for consistency with all other modes. Removed the unreachable `let _ = name_provided;` line at the end of `main()`. Renamed `name_provided` to `_name_provided` (with leading underscore) to suppress the Rust unused-variable warning idiomatically.
