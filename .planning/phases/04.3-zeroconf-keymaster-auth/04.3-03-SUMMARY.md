## What Was Built

### Settings.pm (full rewrite)

1. **OAuth/PKCE removed:** The `if ($paramRef->{startOAuth})` block (lines 60-74 in old file), `clientId` and `redirectUri` template params, and `buildRedirectUri()` call are all gone.

2. **`prefs()` updated:** `clientId` removed from return list. New: `return ($prefs, 'bitrate', 'binary', 'normalization')`.

3. **`new()` extended:** After `SUPER::new`, registers the AJAX endpoint:
   ```perl
   Slim::Web::Pages->addRawFunction(
       'plugins/SpotOn/settings/discoveryStatus',
       \&_discoveryStatusHandler
   );
   ```

4. **`handler()` ZeroConf actions:** `startDiscovery` and `stopDiscovery` form params delegate to `TokenManager->startDiscovery()` / `TokenManager->stopDiscovery()`.

5. **`handler()` template params:** `discoveryRunning = _isDiscoveryRunning() ? 1 : 0` replaces `clientId`/`redirectUri`.

6. **`_discoveryStatusHandler()`:** Raw function handler returning JSON `{status: idle|waiting|connected}` based on credentials.json existence and discovery process state. Uses `to_json`, sets Content-Type application/json.

7. **`_isDiscoveryRunning()`:** Guards against circular dependency by checking `$INC{'Plugins/SpotOn/API/TokenManager.pm'}` before calling `TokenManager->isDiscoveryRunning()`. Returns 0 if TokenManager not yet loaded.

8. **Account management preserved:** `removeAccount` (WR-03 validation: 8-char hex + exists check) and `switchAccount` (WR-04 validation: exists check) blocks are unchanged.

### basic.html (full rewrite of auth section)

- Binary status, bitrate select, normalization checkbox: unchanged
- **PKCE Setup Wizard completely removed:** Client-ID input, OAuth button, developer.spotify.com link, `spoton_auth_pending` sessionStorage, `authUrl` link
- **New ZeroConf Discovery section** (shown when no accounts):
  - connect-hint.png centered, max-width 300px
  - `discoveryRunning` conditional: shows AJAX poll script + Stop Discovery button when active; shows Start Discovery button when idle
  - AJAX poll: XHR GET to `/plugins/SpotOn/settings/discoveryStatus` every 3000ms; on `status === 'connected'` → `clearInterval + location.reload()`
- **Account section:** "Add Another Account" button changed from `name="startOAuth"` to `name="startDiscovery"`
