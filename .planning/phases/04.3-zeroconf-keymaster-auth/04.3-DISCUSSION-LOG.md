## ZeroConf Discovery UX

### Frage 1: Wie soll der User ZeroConf Discovery starten?

| Option | Description | Selected |
|--------|-------------|----------|
| Button in Settings | "Mit Spotify verbinden" Button startet librespot im Discovery-Modus | |
