## Issues Found During UAT (all fixed)

### UAT-FIX-1: Made For You leer — falscher Endpoint
- **Problem:** `_madeForYouFeed` nutzte `me/playlists` + Regex-Filter statt browse/categories
- **Root Cause:** Kuratierte Playlists erscheinen nicht in me/playlists
- **Fix:** Umgestellt auf `getPersonalMixes` (browse/categories/0JQ5DAt0tbjZptfcdMSKl3/playlists)
- **Commit:** b2091a5

### UAT-FIX-2: Bundled-Token permanent 429 (librespot-Default-ID)
- **Problem:** librespot-Default-Client-ID weltweit geteilt → permanent rate-limitiert
- **Root Cause:** SpotOn nutzte `--get-token` ohne `--client-id` für bundled-Flavor
- **Fix:** SPOTON_DEFAULT_CLIENT_ID auf Herger's registrierte App-ID (93aac68fb06348598c1e67734dfaceee) gesetzt, bundled-Flavor nutzt jetzt `--client-id` mit dieser ID
- **Commit:** ab17cce

### UAT-FIX-3: Null-Eintrag in Playlist-Liste
- **Problem:** Spotify API liefert null-Items im playlists-Array → leerer Eintrag oben + 400-Error bei Klick
- **Fix:** `grep { $_ && $_->{id} }` Filter vor _playlistItem
- **Commit:** 92860c0

### UAT-FIX-4: Release Radar / Discover Weekly fehlten
- **Problem:** Spotify liefert 61 Playlists, limit=50 kappte die letzten 11 (darunter Release Radar #54, Discover Weekly #55)
- **Fix:** Pagination — zweiter Request mit offset=50 wenn total > 50
- **Commit:** e25340a
