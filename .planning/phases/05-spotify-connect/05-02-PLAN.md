    Create Plugins/SpotOn/Connect/Daemon.pm (new file, new directory Connect/) adapting Spotty-NG Daemon.pm.

    **Package and base class:** Plugins::SpotOn::Connect::Daemon, extends Slim::Utils::Accessor. Accessors: id, mac, name, cache, _lastSeen, _spotifyId, _proc, _startTimes, _streamStartTimes, _streamMode, _streamPort.

    **Constants:** MAX_FAILURES_BEFORE_DISABLE_DISCOVERY = 3, MAX_INTERVAL_BEFORE_DISABLE_DISCOVERY = 300 (5 min), MAX_STREAM_FAILURES = 5, MAX_STREAM_INTERVAL = 120 (2 min).

    **Imports:** strict, base Slim::Utils::Accessor, File::Spec::Functions catdir, IO::Select, MIME::Base64 encode_base64, Proc::Background (require on-demand inside start()), Slim::Utils::Log, Slim::Utils::Prefs.

    **new($class, $id):** Call SUPER::new(), set mac($id), id($id with colons stripped), init _startTimes and _streamStartTimes as empty arrayrefs, call start(), return self.

    **start():** Get binary path via Plugins::SpotOn::Helper->get(). Get client via Slim::Player::Client::getClient($self->mac). Build device name: use Slim::Player::Sync::syncname($client) for synced non-group players, else $client->name, truncated to 60 chars (D-11, CON-06). Build cache dir using per-account pattern: catdir($serverPrefs->get('cachedir'), 'spoton', $activeAccountId) where activeAccountId comes from $prefs->client($client)->get('activeAccount') || $prefs->get('activeAccount') || ''. Build CLI args array: -c $cacheDir, -n $deviceName, --disable-audio-cache, --player-mac $self->mac, --lms 127.0.0.1:$httpport, --connect. Add --disable-discovery if $prefs->get('disableDiscovery'). Add --lms-auth with base64-encoded $username:$password if LMS has HTTP auth. Call _checkStartTimes(). Create pipe(port_r, port_w). Spawn via Proc::Background->new({die_upon_destroy=>1, stdout=>$port_w}, $helperPath, @helperArgs). Close port_w in parent. IO::Select with 5s timeout on port_r. Parse "stream_port=N" from readline. Close port_r. Set _streamPort and _streamMode(1).

    **_checkStartTimes():** Crash backoff per PATTERNS.md — if more than MAX_FAILURES within MAX_INTERVAL, set disableDiscovery pref. Always push current time.

    **stop():** Kill proc if alive, clear _streamPort.

    **stopForSync():** Kill proc if alive, clear _streamPort, reset _streamStartTimes and _startTimes.

    **pid():** Return _proc->pid if proc exists.

    **alive():** Return 1 if _proc && _proc->alive.

    **uptime():** Return Time::HiRes::time() - last _startTimes entry.

    SpotOn deviations from Spotty-NG per PATTERNS.md: CLI flag is --connect (not --connect-stream), cache dir uses plugin.spoton prefs and spoton subdir, no rmtree of cache on stop(), lms-auth uses same encode_base64 pattern.
  </action>
