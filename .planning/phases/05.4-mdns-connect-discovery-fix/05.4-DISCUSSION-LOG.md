### Frage 1: Wann soll disableDiscovery automatisch zurückgesetzt werden?

| Option | Description | Selected |
|--------|-------------|----------|
| Timer-Cooldown | Nach z.B. 30 Minuten wird disableDiscovery automatisch auf 0 zurückgesetzt und Daemons neu gestartet | |
| LMS-Neustart | disableDiscovery wird bei jedem LMS-Start auf 0 zurückgesetzt | |
