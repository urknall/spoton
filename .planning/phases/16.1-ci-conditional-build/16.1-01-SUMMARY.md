# Phase 16.1 Plan 01: CI Conditional Build Summary

**Conditional Rust build skip via tag-diff change detection and binary reuse from previous release**

## What Was Done

Added change detection and conditional build logic to the CI release workflow so that Rust cross-compilation (8+ minutes for 8 platform targets) is skipped when a tag push contains only plugin file changes (Perl, strings, HTML, config). When skipped, binaries are downloaded from the previous GitHub release instead.

### Task 1: Add change detection and conditional build logic to CI workflow
- **Commit:** a10bf57
- **Changes:**
  - Added `detect-changes` job that compares `librespot-spoton/` directory between current and previous tag using `git diff --name-only`
  - Modified `build` and `build-macos` jobs to require `detect-changes` and only run when `rust_changed == 'true'`
  - Added `reuse-binaries` job that downloads all 7 platform binaries from previous GitHub release and uploads them as artifacts with identical names to the build jobs
  - Modified `release` job to depend on all four upstream jobs (`detect-changes`, `build`, `lipo`, `reuse-binaries`) with `!cancelled() && !failure()` condition
  - `workflow_dispatch` always forces `rust_changed=true` for full builds
  - No previous tag available: forces full build (T-16.1-03 mitigation)

### Task 2: Validate workflow YAML syntax and job dependency graph
- **Result:** All validations passed, no fixes needed
- Validated: YAML syntax, job dependency graph (no cycles, all references valid), artifact name consistency (7 names match between build and reuse paths), GitHub Actions if-condition syntax

## Job Dependency Graph

```
detect-changes
  |-> build (if rust_changed == true) -> [implicit: lipo skipped]
  |-> build-macos (if rust_changed == true) -> lipo
  |-> reuse-binaries (if rust_changed == false)
  |
  v
release (needs all four, runs if !cancelled() && !failure())
```

**Plugin-only release path:** detect-changes -> reuse-binaries -> release (~1 min)
**Rust change release path:** detect-changes -> build + build-macos -> lipo -> release (~8 min)
