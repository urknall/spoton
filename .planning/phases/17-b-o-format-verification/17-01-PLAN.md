<!-- Key format pipeline components the executor must understand -->

From Plugin.pm updateTranscodingTable():
- Restores base son-* pipelines from snapshot on each call
- Injects runtime params (bitrate, cache dir, helper name, normalization)
- OGG Passthrough Guard (line 1461-1468): deletes son-ogg-*-* and soc-ogg-*-* if binary lacks passthrough capability
- Per-player streamFormat logic (line 1472-1499): deletes competing pipelines based on pref value (auto/ogg/pcm/flac/mp3)

From ProtocolHandler.pm formatOverride():
- Returns 'son' for Browse mode, 'soc' for Connect mode
- Calls Plugin->updateTranscodingTable($client) before each track

From custom-convert.conf:
- son pcm * * -- librespot single-track raw PCM output
- son flc * * -- librespot | flac encoder
- son mp3 * * -- librespot | lame encoder
- son ogg * * -- librespot --passthrough (raw Ogg Vorbis)
- soc pcm * * -- Connect pipe passthrough

From Settings.pm:
- streamFormat pref: validated as auto|ogg|pcm|flac|mp3
- Fallback chain: streamFormat -> connectOggOverride -> 'auto'
</interfaces>
