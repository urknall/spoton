<objective>
Remove the `trackInfoURL` method from ProtocolHandler.pm to stop it from completely replacing the standard LMS TrackInfo menu. This is a deletion-only fix for GH #55.

Purpose: LMS's `Slim::Menu::TrackInfo::menu()` checks for `trackInfoURL` first and returns early if found, preventing all standard items (Add to Favorites, play controls, More Info) and all registered info providers from appearing. SpotOn already has a correct `trackInfoMenu` registered via `registerInfoProvider` in Plugin.pm (line 227) that provides identical SpotOn-specific items. Removing the override lets the LMS framework merge standard items with SpotOn items automatically.

Output: Modified ProtocolHandler.pm (80 lines removed), regression test t/14_context_menu.t
</objective>
